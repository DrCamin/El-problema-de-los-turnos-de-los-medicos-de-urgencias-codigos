En este repositorio se encuentran los códigos mencionados en la memoria del Trabajo de Fin de Master titulado:
  "El problema de los turnos de los médicos de Urgencias"
En él, se encuentran:

  - Modelo_basico.gms: Contiene el código a ejecutar en GAMS con el modelo propuesto para el problema simplificado expuesto en la sección 4.2.
  - Modelo_todosmedicos_mes1.gms: Contiene el código a ejecutar en GAMS con el modelo propuesto para el problema real explicado en la memoria
    y modelizado en la sección 4.3.
  - Modelo_todosmedicos_mes1.gms: Contiene el código a ejecutar en GAMS con el modelo propuesto para el problema real en su segundo mes de
    planificación, incluyendo las particularidades necesarias para tener en cuenta el primer mes, como se expone en la sección 5.
  - Modelo_todosmedicos_mes1.gms: Contiene el código a ejecutar en GAMS con el modelo propuesto para el problema real en su tercer mes de
    planificación, incluyendo las particularidades necesarias para tener en cuenta el primer mes, como se expone en la sección 5.
  - Solutions_to_excel.ipynb: Contiene el código en Python mencionado al inicio de la sección 5. Al ejecutarse, este código toma el archivo
    .lst que ofrecen las ejecuciones de los programas anteriores como solución, y devuelve un archivo de Excel con la planificación de la
    solución desarrollada y una tabla con datos acerca del reparto.
